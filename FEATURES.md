# 功能特性详解 / Features Overview

## 🎨 视觉设计特性

### 1. 现代科技感配色
```
主题色彩：
├── 深色背景: #0A0E27 (科技感深蓝黑)
├── 青色强调: #00D9FF (未来感青色)
├── 紫色强调: #7B2FF7 (神秘紫色)
└── 渐变效果: 青色→紫色 (135度)
```

**应用场景**：
- Hero区域背景
- 按钮和CTA元素
- Logo和标题文字
- 卡片边框装饰
- 悬停效果

### 2. 玻璃态设计 (Glassmorphism)
```css
backdrop-filter: blur(20px);
background: rgba(255, 255, 255, 0.08);
border: 1px solid rgba(255, 255, 255, 0.18);
```

**应用位置**：
- 导航栏
- 卡片容器
- 联系信息框
- 二维码容器

### 3. 动态背景效果

#### 粒子动画
- 浮动的光点效果
- 20秒循环动画
- 多层次视觉深度

#### 网格背景
- 科技感网格线
- 半透明青色
- 50px × 50px 网格

#### 渐变流动
- 15秒旋转动画
- 径向渐变效果
- 自动循环播放

### 4. 动画系统

#### 淡入动画 (fadeInUp)
```css
从下方30px淡入
持续时间: 1秒
缓动函数: ease-out
```

**应用元素**：
- Hero标题和文字
- 服务卡片
- 页面内容区

#### 悬浮动画 (float)
```css
上下浮动20px
持续时间: 6秒
无限循环
```

#### 脉冲动画 (pulse)
```css
发光扩散效果
持续时间: 2秒
无限循环
```

#### 滚动动画 (scrollDown)
```css
滚动指示器
上下移动15px
2秒循环
```

## 🎨 设计模式

### 卡片设计
```
特点：
├── 圆角: 12-16px
├── 阴影: 多层次
├── 边框: 渐变或透明
├── 悬停: 上移+发光
└── 过渡: 0.4秒
```

### 按钮设计

## 📊 浏览器兼容性

### 完全支持
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

### 部分支持
- ⚠️ IE 11 (不支持backdrop-filter)
- ⚠️ 旧版移动浏览器

### 降级方案
- 不支持backdrop-filter时使用纯色背景
- 不支持CSS Grid时使用Flexbox
- 减少动画模式自动禁用动画
