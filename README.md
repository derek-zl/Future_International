# Future International Co., Ltd. - 官方网站

## 🚀 重构亮点

### 现代科技感设计
- **渐变色彩系统**：采用青色(#00D9FF)到紫色(#7B2FF7)的现代渐变配色
- **玻璃态设计**：毛玻璃效果(backdrop-filter)营造高级质感
- **动态背景**：粒子动画、网格背景、渐变流动效果
- **流畅动画**：淡入、悬浮、脉冲等多种动画效果

### 技术特性
- ✅ 完全响应式设计，完美适配移动端
- ✅ 中日英三语切换，翻译准确完整
- ✅ 平滑滚动和页面过渡动画
- ✅ 交互式导航菜单（移动端汉堡菜单）
- ✅ 表单验证和提交处理
- ✅ 性能优化的CSS动画

## 🌐 多语言支持

网站支持三种语言，通过点击右上角语言切换按钮：
- **EN** - English (英语)
- **JP** - 日本語 (日语)
- **CN** - 中文 (简体中文)

语言偏好会自动保存到浏览器本地存储。

## 📱 响应式断点

- **桌面端**: > 992px
- **平板端**: 768px - 992px
- **移动端**: < 768px

## 📝 使用说明

1. 直接在浏览器中打开 `index.html`
2. 点击右上角切换语言
3. 导航菜单可访问各个页面
4. 移动端点击汉堡菜单展开导航
