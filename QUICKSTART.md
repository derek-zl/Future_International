# 快速启动指南 / Quick Start Guide

## 🚀 立即查看网站

### 方法1: 直接打开（推荐）
1. 双击 `index.html` 文件
2. 网站将在默认浏览器中打开

### 方法2: 使用本地服务器
```bash
# 如果安装了Python 3
python3 -m http.server 8000

# 如果安装了Node.js
npx serve

# 然后访问: http://localhost:8000
```

## 📱 测试功能

### 1. 语言切换
- 点击右上角的 **EN / JP / CN** 按钮
- 页面内容会立即切换语言
- 语言偏好会自动保存

### 2. 导航测试
- 点击顶部导航菜单
- 测试所有页面链接：
  - Home (首页)
  - About Us (关于我们)
  - Services (服务)
  - Contact (联系我们)

### 3. 移动端测试
- 缩小浏览器窗口到手机尺寸
- 或按 F12 打开开发者工具，切换到移动设备模式
- 点击汉堡菜单图标（三条横线）
- 测试移动端导航菜单

### 4. 交互效果
- 鼠标悬停在服务卡片上
- 滚动页面查看动画效果
- 点击"Contact Us"按钮
- 填写并提交联系表单

## 🔧 自定义修改

### 修改配色
编辑 `style.css` 文件中的 CSS 变量：
```css
:root {
    --accent-color: #00D9FF;        /* 主强调色 */
    --accent-secondary: #7B2FF7;    /* 次强调色 */
    --bg-dark: #0A0E27;             /* 深色背景 */
    /* ... 更多颜色 */
}
```

### 修改文字内容
编辑 `script.js` 文件中的 `translations` 对象：
```javascript
const translations = {
    'en': { /* 英文翻译 */ },
    'jp': { /* 日文翻译 */ },
    'cn': { /* 中文翻译 */ }
};
```

### 修改公司信息
在各个 HTML 文件中搜索并替换：
- 公司名称
- 地址
- 联系方式
- 业务描述

## 💡 常见问题

### Q: 为什么动画不流畅？
A: 确保使用现代浏览器，并关闭浏览器的"减少动画"设置。

### Q: 如何更换图片？
A: 替换 `assets/images/` 文件夹中的图片，或修改HTML中的图片链接。

### Q: 语言切换不工作？
A: 检查浏览器控制台是否有JavaScript错误，确保 `script.js` 正确加载。

### Q: 移动端菜单无法打开？
A: 清除浏览器缓存，刷新页面重试。

## 📞 技术支持

如有问题，请检查：
1. 浏览器控制台（F12）的错误信息
2. 确保所有文件在同一目录
3. 检查网络连接（字体加载需要网络）
